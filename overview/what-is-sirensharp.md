# ❓ What is SirenSharp

SirenSharp is a tool which is used to generate FiveM resources that allows for custom server sided sounds to be streamed to clients.

<figure><img src="../.gitbook/assets/image (6).png" alt=""><figcaption></figcaption></figure>

It allows you to create multiple AWC packs with many sirens in each. This is a great way to split up sirens between emergency departments for example.
