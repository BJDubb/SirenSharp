# 📂 Exporting Resource

Once you are happy with your project, press the `Tools` button in the menu bar and hit `Generate FiveM Resource...`

<figure><img src="../.gitbook/assets/image (1).png" alt=""><figcaption></figcaption></figure>

This will open up a window which will allow you to export the sirens into a FiveM resource. The `DLC name` will need to be set. It needs to be unique and can't match any other SirenSharp DLC's you may have created. Once you have done that, hit `Create` and your resource will be generated in the location specified.

<figure><img src="../.gitbook/assets/image (3).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Make sure you note down the DLC name you enter as you will need it when loading sounds in FiveM.
{% endhint %}
