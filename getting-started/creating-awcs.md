# 🔊 Creating AWC's

## What is an AWC?

An Audio Wave Container (AWC) is a special type of file used by GTA 5 and FiveM to store `.wav` files and play them at game runtime. It is essentially a folder with a bit of metadata that holds your sounds and related data to the sounds.



## How do I create one?

Click on the `AWC` tab in the menu bar and press `New`

<figure><img src="../.gitbook/assets/image (4).png" alt=""><figcaption></figcaption></figure>

This will create a new AWC in the AWC List. Click on the AWC to see info about it and the sirens it holds. Here you can change the name of your AWC.&#x20;

{% hint style="info" %}
AWC names need to follow a few rules. See here for those rules.
