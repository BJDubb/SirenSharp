# 🚨 Creating Sirens

Once you have [created an AWC](creating-awcs.md), you can create sirens that belong to these AWC's. With an AWC selected, hit the `Siren` button in the menu bar and you will be met with 2 options.

<figure><img src="../.gitbook/assets/image (7).png" alt=""><figcaption></figcaption></figure>

Pressing `New` will create a new siren with a generic name and no audio file associated with it.

<figure><img src="../.gitbook/assets/image (9).png" alt=""><figcaption></figcaption></figure>

Pressing `Import WAV...` will open up a file picker dialog which will allow you to select multiple audio files. The sirens will have their name generated based on the file name and the audio path will be assigned.<br>

<figure><img src="../.gitbook/assets/image (10).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../.gitbook/assets/image (2).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
**NOTE:** Any audio files used **MUST** be in the .WAV format and must be a 16bit PCM Mono audio. SirenSharp will fail to generate sirens for incorrect formats.
{% endhint %}
